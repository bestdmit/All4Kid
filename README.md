# All4Kid

**Единая платформа для поиска и онлайн-записи ко всем детским специалистам**

 <!-- ![All4Kid Logo](путь к логотипу) -->

## О проекте

All4Kid — это удобный и современный онлайн-агрегатор, который объединяет всех специалистов для детей в одном месте. От педиатров и логопедов до тренеров по плаванию и аниматоров — мы помогаем родителям быстро находить проверенных профессионалов и записываться к ним онлайн.

**Целевая аудитория:** Родители детей в возрасте от 0 до 16 лет, проживающие в крупных городах.
